# Job Market Data Pipeline

An end-to-end data engineering project built to learn and demonstrate
ETL fundamentals using real-world job market data.

## Project Goal
- Learn data engineering by building a practical pipeline
- Ingest raw job data
- Clean and normalize the data
- Prepare it for analytics and databases
